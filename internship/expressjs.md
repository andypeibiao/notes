# Express
Fast, unopinionated, minimalist web framework for node.

## 安装
 创建package.json
```
$ npm init
```
```
$ npm install express --save
```

## Express 应用生成器

```
$ npm install express-generator -g
```
```
$ express -h
```
```
$ express myapp
```
```
$ cd myapp
$ npm install

#启动这个应用（MacOS 或 Linux 平台）：

$ DEBUG=myapp npm start
```
