## 输出数据
print()不是函数而是语言结构，返回值都是1
与echo相比，echo无返回值速度稍快，并无太大差异

printf()打印静态文本和变量组成的信息（类似C语言） 可以控制类型，精度，对齐方式和位置

sprintf()和printf()功能相同
差异在于，printf()可直接呈现在浏览器上
```
$cost = sprintf("$%.2f",43.2); //$cost = $43.20
```
